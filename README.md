# Kuznecov.io

Моё резюме на Vue, Express с поддержкой SSR — [kuznecov.io](https://kuznecov.io/)

## Build Setup 

``` bash
# install dependencies
npm install # or yarn

# serve in dev mode
npm run dev

# build for production
npm run build

# serve in production mode
npm run start
```
